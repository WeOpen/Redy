# Redy 🦝✨

> WeOpen 的温暖开源守护者，也是这个品牌展示站点的主角。

## 🌟 项目简介

- 这是一个基于 Next.js 构建的品牌展示站点
- 用来承载 Redy 的官网表达、视觉资产与品牌故事
- 支持 GitHub Pages 自动发布

## 🧩 系统内容

- 品牌故事与角色设定
- 官网单页展示
- 品牌资源目录
- 基础测试
- 自动部署工作流

## 🗂️ 目录结构

```text
app/            页面与界面
brand/          Logo、吉祥物、图标等品牌资产
tests/          测试文件
DraftDesign.md  原始品牌设计草案
```

## 🚀 本地开发

```bash
npm install
npm run dev
```

## 🧪 运行测试

```bash
npm test
```

## 🌍 GitHub Pages

推送到 `main` 后会自动构建并发布到 GitHub Pages。

预计访问地址：`https://weopen.github.io/Redy/`
